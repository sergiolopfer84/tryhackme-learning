## Summary

This room focused on system maintenance, automation, package management, and log analysis, introducing concepts essential for real-world Linux administration and security monitoring.

## Key concepts learned

- Managing and editing files using terminal-based text editors.
- Understanding Linux processes and basic system monitoring.
- Automating tasks using cron jobs and correctly interpreting crontab schedules.
- Managing software with APT, including repositories, package installation, updates, and removal.
- Adding and removing third-party repositories and understanding the role of GPG keys.
- Exploring system and service logs located in `/var/log`.
- Analysing Apache web server logs to identify client IP addresses and accessed resources.
- Understanding log rotation and how file permissions affect log accessibility.

## Security relevance

The topics covered in this room are directly applicable to system administration, incident response, and security operations.  
Understanding automation mechanisms, package trust, permissions, and log analysis is essential for detecting misconfigurations, investigating incidents, and maintaining secure Linux environments.
